j2g — JSON to Go Struct Converter

Usage:
  j2g [--name <StructName>] < input.json > output.go

Flags:
  --name <StructName>   Name of the root Go struct (default: Result)
  -h, --help            Show this help message

Description:
  j2g reads JSON data from standard input and writes the corresponding Go struct
  definition to standard output. It's useful for quickly generating Go types
  based on example JSON responses or payloads.

Example:
  cat data.json | j2g
  cat data.json | j2g --name Response > model.go
